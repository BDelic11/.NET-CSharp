# ️ PhoneBook Console App (C# / .NET)

A simple console application built with **C#** and **.NET** to manage personal contacts and phone call records — designed for practicing **OOP**, **SOLID principles**, and clean application architecture.

---

##  Overview

This project simulates a personal phonebook where users can:

- Manage contacts (add, edit, delete)
- Create and track outgoing phone calls
- Handle call statuses (InProgress, Missed, Completed)
- Navigate via a console-based menu and submenu
- Practice layered architecture and dependency injection

---

##  Key Concepts Practiced

- ✅ Object-Oriented Programming (OOP)
- ✅ Clean Architecture (Separation of Concerns)
- ✅ SOLID Principles
- ✅ Interface-based design
- ✅ Service-repository pattern
- ✅ Dependency Injection (manual)
